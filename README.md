## Files

I file markdown possono essere compilati in PDF con [pandoc](https://pandoc.org/) (serve aver installato [TeX](https://tug.org/texlive/)!)

```sh
pandoc --metadata-file pandoc_metadata.yaml <file_in>.md -o <file_out>.pdf
```
